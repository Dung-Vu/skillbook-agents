## 📖 Why Do We Need This Skill?

Creating professional videos from scratch demands complex technical skills. This skill enables the agent to automatically convert text scenarios or static images into storyboards and generate smooth, professional motion video sequences.

## ⚙️ How It Works

The text/image-to-video workflow operates as follows:

```
[Prompt/Image Input] -> [Storyboarding & Scripting] -> [Frame Selection] -> [Video Generation Engine] -> [Output Video Asset]
```

1. **Planning**: Drafts detailed scripts and breaks them down into scenes.
2. **Prompt Engineering**: Crafts optimized motion prompts (camera moves, lighting changes) for the video generation model.
3. **Rendering**: Triggers video generation and runs basic asset verification.

## 🚀 Agent Guidelines (Prompt Guidelines)

1. Explicitly define the target aspect ratio (16:9 for widescreen, 9:16 for vertical platforms like TikTok/Reels).
2. Write clear camera motion descriptors (e.g., slow pan right, steady zoom in) to improve cinematic feel.
3. When animating static images, preserve the core structure to avoid severe visual artifacts.

## ⚠️ Gotchas and notes

- **Motion Distortion**: Excessive motion vectors can cause object morphing. Keep initial motion values moderate.
- **Style Inconsistency**: When generating multi-scene videos, maintain consistent stylistic prompts across all segments.
