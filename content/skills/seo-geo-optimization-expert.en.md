## 📖 Why Do We Need This Skill?

AI search platforms (Perplexity, Gemini, SearchGPT) aggregate answers and cite authoritative sources. Generative Engine Optimization (GEO) is essential to secure these citations. This skill provides rules to enhance content readability for LLMs, deploy structured data schemas, and optimize readability indexes.

## ⚙️ How It Works

The SEO/GEO audit workflow:
```
Content Analysis --> GEO Citation Scoring --> EEAT & Vocabulary Optimization --> Schema & Deliverables Export
```
1. **Content Ingestion**: Ingest raw articles, landing pages, or product descriptions.
2. **Citation Audit**: Evaluate the text against GEO drivers (cite sources, statistical proof, quotation density, readability).
3. **EEAT Hardening**: Inject trust markers, expert quotes, first-party data, and define concepts sharply.
4. **Technical Schema**: Generate corresponding JSON-LD schemas (Article, FAQ, LocalBusiness) to assist crawling.

## 🚀 Agent Guidelines (Prompt Guidelines)

- Structure answers using direct definitions ('[Term] is a [category] that...') to facilitate LLM extraction.
- Implement strict schema validation—ensure all generated JSON-LD scripts are syntactically valid and parseable.
- Support statements with data points: replace generic descriptions with specific statistics and authoritative citations.

## ⚠️ Gotchas and notes

- **Keyword Stuffing**: Do not repeat target terms unnaturally. AI crawlers favor readability and semantic richness over keyword frequency.
- **Fictional references**: Never generate false statistics or insert broken URLs for citation credibility.
