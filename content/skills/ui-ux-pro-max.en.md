## 📖 Why Do We Need This Skill?

Web aesthetics require sophisticated palettes and custom typography rather than typical generic framework layouts. This skill empowers the Agent to select premium font pairings, establish color tokens (such as warm earth, cold slate, or deep monochrome), and deliver clean, modular CSS modules.

## ⚙️ How It Works

The high-fidelity design process:
```
Brand Concept Ingestion --> Color Palette & Font Pairing --> Modular CSS Architecture --> Interaction Polish
```
1. **Concept Definition**: Map the desired brand character (e.g., Luxury Minimalist, Creative Modern, High-Tech Dark).
2. **Design Synthesis**: Select custom hex palettes and pair headers (e.g. Playfair Display) with body fonts (e.g. Inter).
3. **Coding**: Package variables and CSS tokens inside clean, semantic, and reusable CSS stylesheets.
4. **Polish**: Tune micro-interactions, hover states, transitions, and loading states for a premium feel.

## 🚀 Agent Guidelines (Prompt Guidelines)

- Define exact custom color palettes. Avoid standard framework defaults.
- Always use highly contrastive typography pairings (e.g., serif headings with sans-serif body copy).
- Provide clean, copy-pasteable CSS/HTML structures utilizing CSS Custom Properties for theme customizability.

## ⚠️ Gotchas and notes

- **Font Bloat**: Limit project font weights and families to a maximum of 2 to preserve fast load times.
- **Overdone Transitions**: Restrict transition durations to 150-300ms. Avoid complex, heavy animations that degrade mobile UX.
